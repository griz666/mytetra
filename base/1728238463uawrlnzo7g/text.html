Панель esphome всегда показывает статус устройств как offline

поскольку разные подсети то надо явным образом на роутере припиывать статический dns для устройств.

Вообще есть рекомендация:

" ESPHome uses mDNS to show online/offline state in the dashboard view. So for that feature to work you need to enable host networking mode


mDNS might not work if your Home Assistant server and your ESPHome nodes are on different subnets. If your router supports Avahi, you are able to get mDNS working over different subnets.


Just follow the next steps:


Enable Avahi on both subnets.

Enable UDP traffic from ESPHome node’s subnet to 224.0.0.251/32 on port 5353.

"



"ESPHome использует mDNS для отображения состояния «онлайн»/«офлайн» на панели мониторинга. Поэтому для работы этой функции вам необходимо включить режим сетевого хоста mDNS может не работать, если ваш сервер Home Assistant и ваши узлы ESPHome находятся в разных подсетях. Если ваш маршрутизатор поддерживает Avahi, вы можете заставить mDNS работать в разных подсетях. Просто выполните следующие шаги: Включите Avahi в обеих подсетях. Включите UDP-трафик из подсети узла ESPHome в 224.0.0.251/32 на порту 5353."
