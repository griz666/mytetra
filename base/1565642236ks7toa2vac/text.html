Для просмотра текущего imei :

- AT*MRD_IMEI=R

 

 

Для смены imei на R11e-LTE :

 

- Удаляем текущий imei командой AT*MRD_IMEI=D

- Прописываем новый в формате AT*MRD_IMEI=W,0101,11JAN1970,000000000000000

 

Команды вводим через терминал например

- /interface lte at-chat lte1 input="AT*MRD_IMEI=R"
