Вариант с Network Manager

First, find the name of the connection profile in question by looking at the output of nmcli connection.

Then, modify the profile accordingly, like nmcli connection modify "$PROFILE_NAME" +ipv4.routes "192.268.7.0/24 192.168.7.1"

Look at the result: nmcli connection show "$PROFILE_NAME".

Finally, reactive the profile with nmcli connection up "$PROFILE_NAME".

Check the result with ip route show

You can also use any other NetworkManager client, like the GUI nm-connectioneditor`.
