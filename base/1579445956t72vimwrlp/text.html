В качестве KVM хостов выступают CentOS, предположим с хоста А необходимо переместить машину на хост Б.

На хосте А
Делаем дамп конфигурации переносимой машины машины:

virsh dumpxml myserver > /mnt/data-1/myserver.xml

Переносим xml и образ(ы) жесткого диска на хост Б

На хосте Б
Копируем xml файл в папку:

/run/libvirt/qemu

Определяем машину:

virsh define /run/libvirt/qemu/myserver.xml

При необходимости правим (указываем месторасположение образа жесткого диска):

virsh edit myserver

З.Ы. myserver - это имя виртуальной машины )
