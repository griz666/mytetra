Проблема - при работающем Kodi монитор гаснет без активности

надо сделать так

export DISPLAY=:0

sudo xset -dpms

sudo xset s off

и поставить в астозапуск, например ~/.bashrc

проверить

xset q


Второ способ, в xorg.conf

Section «ServerFlags»
Option «BlankTime» «0»
Option «StandbyTime» «0»
Option «SuspendTime» «0»
Option «OffTime» «0»
EndSection


важно что параметр тоже в кавычках
