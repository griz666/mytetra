Видеокарта по умолчанию не разрешает упралении скоростью вентилятора.

лечится так

echo 1 > /sys/class/drm/card0/device/hwmon/hwmon0/pwm1_enable


но не работает при выходе из спящего режима


Для этого надо такой же скрипт положить здесь

/lib/systemd/system-sleep



#!/bin/sh


case $1 in

post)

echo 1 > /sys/class/drm/card0/device/hwmon/hwmon0/pwm1_enable

;;

esac
